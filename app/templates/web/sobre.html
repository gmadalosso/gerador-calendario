Sobre o Gerador de Calendário

O que é?

O Gerador de Calendário é uma ferramenta web que permite criar calendários personalizados em formato PDF, prontos para impressão. Você pode escolher o ano, a orientação da página, incluir feriados nacionais e estaduais, fases da lua, contagem de semanas e dias do ano.

Funcionalidades

  • Calendários personalizados: Escolha o ano desejado (1900-2200)
  • Orientacao flexível: Vertical (retrato) ou horizontal (paisagem)
  • Feriados: Feriados nacionais e opcionalmente feriados estaduais
  • Fases da lua: Visualize as fases da lua diretamente no calendário
  • Contagem de semanas: Opção para mostrar o número da semana
  • Dia do ano: Veja a contagem de dias do ano para cada data
  • Download em PDF: Gere e baixe seu calendário em formato PDF

Aplicação independente de serviços externos

Esta aplicação não depende de APIs externas. Todos os cálculos são feitos localmente no servidor, incluindo:

  • Cálculo automático da data da Páscoa e feriados móveis (Carnaval, Sexta-feira Santa, Corpus Christi)
  • Feriados nacionais fixos armazenados localmente
  • Cálculo preciso das fases da lua usando algoritmos astronômicos

Isso garante que a aplicação funcione para qualquer ano (1900-2200) sem depender de serviços externos.

Tecnologias

Esta aplicação foi desenvolvida usando:

  • FastAPI - Framework web moderno e rápido para Python
  • WeasyPrint - Geração de PDFs a partir de HTML/CSS
  • Bootstrap 5 - Framework CSS para interface responsiva
  • PyEphem - Cálculo preciso das fases da lua
  • Algoritmo de Meeus/Jones/Butcher - Cálculo da data da Páscoa
